Project Overview

This project allows:

Dealers to add vehicles and create auctions.

Users to browse vehicles and place bids in active auctions.

Super Admin to manage dealers, auctions, and monitor bids.

Notifications for dealer approvals via email.

Real-time bidding updates (optional WebSockets support).

Technologies used:

Python 3.12

Django 5

Django REST Framework (DRF)

Django Channels (optional, for real-time bidding)

PostgreSQL 

Redis (optional, for Channels)

Postman (for API testing)
